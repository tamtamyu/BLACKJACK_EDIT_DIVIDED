# Blackjack (Qt/C++)

Qt Creator と C++ を用いて作成したGUIブラックジャックゲームです。

## 概要

このゲームではブラックジャックを遊ぶことができます。
掛け金を決め、10$を20$にしたらあなたの勝ちです。

## 主な機能

- GUIでプレイ可能
- ベット機能
- ディーラーAI
- カード画像表示

## 開発環境

- C++17
- Qt 6
- CMake

## ビルド方法

1. リポジトリをクローンします。

```bash
git clone https://github.com/tamtamyu/BLACKJACK_EDIT_DIVIDED
```

2. プロジェクトを **Windows上のフォルダ**（例: Desktop や Documents）に配置します。
   ※ WSL（`/wsl.localhost/...`）上では正常にビルドできない場合があります。

3. Qt Creatorで `CMakeLists.txt` を開きます。

4. Configure Project を実行します。

5. Build（Ctrl + B）

6. Run（Ctrl + R）
