# Blackjack (Qt/C++)

Qt Creator と C++ を用いて作成したGUIブラックジャックゲームです。

## 概要

このゲームではブラックジャックを遊ぶことができます。
掛け金を決め、10$を20$にしたらあなたの勝ちです。

## 主な機能

- GUIでプレイ可能
- ベット機能
- ディーラーAI
- カード画像表示

## 開発環境

- C++17
- Qt 6
- CMake

## ビルド方法

1. Qt Creatorで `CMakeLists.txt` を開く
2. Configure
3. Build
4. Run
